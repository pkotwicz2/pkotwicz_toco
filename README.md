# Gemini Joke Injector Chrome Extension (Off-Device)

This extension targets pages under the `https://pkotwicz2.github.io/` origin. On load, it queries the external Gemini API (Gemini 1.5 Flash) for a joke and inserts it as an `<h1>` at the start of the document.

## Files
- `manifest.json`: Extension manifest (MV3).
- `content.js`: Content script injected into matching pages.
- `background.js`: Service worker handling Gemini API calls.
- `config.js`: Configuration file containing the Gemini API key (ignored by git).
- `.gitignore`: Ensures `config.js` is not committed.

## Prerequisites

You need a Gemini API key to use this extension.

1. Get an API key from [Google AI Studio](https://aistudio.google.com/).
2. Create or edit `config.js` in the extension directory.
3. Set your API key in `config.js`:
   ```javascript
   globalThis.GEMINI_API_KEY = 'YOUR_ACTUAL_API_KEY';
   ```

## Installation Instructions

1. Open Google Chrome.
2. Navigate to `chrome://extensions/`.
3. Enable **Developer mode** (toggle switch in the top right).
4. Click **Load unpacked** (button in the top left).
5. Select the directory containing these files:
   `/usr/local/google/home/pkotwicz/Desktop/pkotwicz_toco`

## Testing

1. Navigate to any page under `https://pkotwicz2.github.io/`.
2. The page should load and a joke generated by Gemini 1.5 Flash should appear as a large heading at the top of the page.
